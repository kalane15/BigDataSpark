# BigDataSnowflake
Анализ больших данных - лабораторная работа №2.

Иванченко Макар Дмитриевич, М8О-310Б-23.

Запуск:

docker compose up -d --build

Джобы спарка запустятся автоматически при сборке контейнера. Надо подождать, пока в логах контейнера со спарком не напишется "All ETL jobs completed".
Пароли, пользователи и порты для подключения к бд можно найти в main.py.
У таблиц и файлов есть префикс dm<номер>, показывающий, к какой витрине они относятся (номера витрин как в задании).
